# コーディングのテンプレート

cloneして使えば、コーディングの時間短縮になります。

## 使い方

`$ git clone https://github.com/AyakaKusakari/coding-template.git`

`$ cd coding-template`

`$ npm install`

`$ gulp`

## 変更箇所をpushする方法

`$ git clone https://github.com/AyakaKusakari/coding-template.git`

ローカルで編集

`$ cd coding-template`

`$ git add .`

`$ git add ****.html(ファイル名)`

`$ git commit -m "変更箇所"`

`$ git push`
